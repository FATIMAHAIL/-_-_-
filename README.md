🌙 مولد بطاقات تهنئة رمضان | Ramadan Greeting Card Generator
تطبيق ويب بسيط وسريع يتيح للمستخدمين إنشاء بطاقات تهنئة رمضانية مخصصة عن طريق رفع صورة وكتابة الاسم عليها مباشرة من المتصفح.

A simple and fast web application that allows users to create customized Ramadan greeting cards by uploading an image and adding their names directly from the browser.

✨ المميزات (Features)
رفع صور مخصصة: يمكنك اختيار أي قالب بطاقة تريده من جهازك.

إضافة نص ذكي: يقوم التطبيق بكتابة الاسم في مكان محدد وتلقائي على الصورة.

معاينة مباشرة: عرض البطاقة فوراً قبل تحميلها.

تحميل بجودة عالية: إمكانية حفظ البطاقة بصيغة PNG وبدقة ممتازة.

متجاوب (Responsive): تصميم عصري يتناسب مع الهواتف والأجهزة اللوحية.

🚀 التقنيات المستخدمة (Technologies Used)
HTML5: لهيكلة الصفحة.

CSS3: للتصميم الجمالي واستخدام خط "Alexandria" من Google Fonts.

JavaScript (Canvas API): لمعالجة الصور والكتابة عليها برمجياً دون الحاجة لخادم (Client-side).

🛠️ كيف يعمل الكود؟ (Technical Overview)
يعتمد المشروع على عنصر <canvas> في HTML5. إليك الخطوات التقنية:

يتم قراءة الملف المرفوع باستخدام FileReader.

يتم رسم الصورة على الكانفاس بأبعادها الأصلية لضمان الجودة.

يتم حساب حجم الخط ومكانه بناءً على أبعاد الصورة (نسبة مئوية) لضمان تناسق النص.

يتم تحويل الكانفاس إلى DataURL لعرض المعاينة وتحميل الصورة.

📖 طريقة الاستخدام (How to Use)
قم بتحميل الملفات أو عمل Clone للمستودع.

افتح ملف index.html في أي متصفح.

اختر صورة البطاقة، اكتب الاسم، واضغط على "توليد البطاقة".
